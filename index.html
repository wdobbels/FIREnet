Interactive plots

Far-InfraRed Emission Networks (FIRE-net) is a machine learning framework that aims to estimate the far-infrared (FIR) spectral energy distribution (SED) of a galaxy, based on the ultraviolet to mid-infrared (UV–MIR) SED.

Below you can find the interactive figures for the FIREnet project. The main github page, with its scripts and jupyter notebooks, can be found here. The accompanying paper, "Predicting the global far-infrared SED of galaxies via machine learning techniques", will be linked here once accepted.

Click on the tabs below to switch figures. Click on one of the data points to select a particular galaxy, which then also shows its SED in the plot below. Clicking on the legend elements allows you to show/hide elements.
